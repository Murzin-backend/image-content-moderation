# image-content-moderation
Сервис анализа изображений для модерации контента. Определяет наблюдаемые визуальные признаки: сцены взаимодействия между людьми (поцелуи, объятия), откровенный контент, символику и текст на изображениях. Возвращает объяснимые результаты (метки и вероятности) через API. Поддерживает запуск в Docker и использование GPU.
